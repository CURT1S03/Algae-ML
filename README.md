# Algae-Bloom-ML-4641
/data/: Description of the directory

/data/final_data_synthetic.csv: Description of the file. Data folder contains final_data_synthetic.csv (including 600 samples from our initial dataset, 400 samples from our Lake Mendota dataset, and 4000 synthetically generated samples.

/data/final_dataset.csv: Data folder contains final_dataset.csv: combination of 600 samples and 400 samples without synthetic.

/data/Iowa.csv: Iowa.csv: 600 samples from our Iowan lake dataset.

/data/ntl439_1_v1.csv: 400 samples from Lake Mendota dataset.

/modelsave/: Model save folders contain models with saved parameters for future reusage.

/modelsave/model_elasticnet.joblib: model_elasticnet.joblib job lib file for elasticnet model.

/modelsave/model_hgbr.joblib: model_hgbr.joblib job lib file for hgbr model.

/modelsave/model_pca_xgb.joblib: model_pca_xgb.joblib joblib file for pca+xgb model.

/notebooks/: Notebooks folder contains the ipynb file used (codebase) for our project.

/notebooks/ipynb_checkpoints/: ipynb_checkpoints folder within notebooks folder contains saved versions of the checkpoint model.

/notebooks/ipynb_checkpoints/model-checkpoint.ipynb: model-checkpoint.ipynb is a checkpoint version of the model file where we do our model building and visualization.

/notebooks/ipynb_checkpoints/preprocessing-checkpoint.ipynb: preprocessing-checkpoint.ipynb is a checkpoint version of the preprocessing file.

/notebooks/model.ipynb: Inside notebooks but not within the ipynb_checkpoints folder is model.ipynb which is the current versin of our model notebook.

/notebooks/preprocessing.ipynb: Inside notebooks but not within the ipynb_checkpoints folder is preprocessing.ipynb which is the current versin of our preprocessing notebook.

/GanttChart.xlsx: Within our Algae-Bloom-ML-4641 but not within any of the inside folders (data, modelsave, notebooks) is the GanttChart for attributing work.

/index.html: index.html to build the GitHub page.

/README.md: README.md file for a description of the repository.

/model-performance-midterm.png: image of model performance using r2, rmse, mae

/prediction-error-distribution-miterm.png: image of the actual versus predicted distribution

/scatterplot.png: image of a scatterplot of actual versus predicted

/z-score-distribution.png: image of the z score distribution
